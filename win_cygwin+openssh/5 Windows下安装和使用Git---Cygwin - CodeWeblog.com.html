5 Windows下安装和使用Git---Cygwin

1）安装Cygwin

从官网 http://www.cygwin.com下载安装程序。

http://cygwin.com/setup-x86_64.exe

默认安装的Cygwin没有安装Git软件包。

可以安装后再次运行安装程序setup.exe.到了下面这个界面后搜索git.


点击All后面的Default。意味着全部安装

git-completion,openssh,vim.

4）用户主目录不一致的问题

如果其他软件在Windows下设置了HOME环境变量，会影响到Cygwin用户主目录的设置。

方法1：

修改 C:\cygwin64\Cygwin.bat

在第一行增加 set HOME=

方法2：

如果希望使用HOME环境变量指向的主目录，可通过手工编辑/etc/passwd文件

将其中的用户主目录修改成HOME环境变量所指向的目录。

5）命令行补齐忽略文件名大小写

编辑文件~/.inputrc,在其中添加设置

set completion-ignore-case on

6)忽略文件权限的可执行位

可以通过下面的配置禁止 Git对文件权限的追踪：

git config --system core.fileMode false

7)Cygwin下Git的中文支持

Cygwin默认使用UTF-8字符集，并能巧妙的与Windows系统的字符集进行转换。

查看字符集的命令locale:


如果中文文件名在Git命令输出的显示有问题

git config --global core.quotepath false
