RedHat上SSH2的安装和使用

作 者： 周涛


How to Run SSH2 on RedHat 6.2

Tao Chou （周涛）
[email protected]

本文件说明如何在 Linux 上安装、使用 SSH2.


一、版权说明

本文件将遵循GPL发布

二、软件取得

1 RedHat 6.2

http://freesoft.online.sh.cn/

2 SSH2 Server

rpm : ssh-commercial-server-2.2.0-2nox.i386.rpm
source: ssh-2.2.0.tar.gz

http://www.ssh.com

3 SSH2 Client
rpm : ssh-commercial-workstation-2.2.0-2.i386.rpm
http://www.ssh.com


三、软件安装

说明：Remote Host Name :Remote (Server) 192.168.1.222
Local Host Name :Local (Client) 192.168.1.223

1 安装 Redhat6.2

2 安装 SSH2 Server(Remote Host)

使用 RPM
[[email protected] root]# rpm -ivh ssh-commercial-server-2.2.0-2nox.i386.rpm

使用 SOURCE

[[email protected] root]# tar xzvf ssh-2.2.0.tar.gz
# cd ssh-2.2.0
# ./configure
# make
# make install


3 安装 SSH2 Clinet(Local Host)

[[email protected] root]# rpm -ivh ssh-commercial-workstation-2.2.0-2.i386.rpm


四、系统配置
1.在 Remote Host 启动sshd
[[email protected] root]/usr/loca/sbin/sshd2

2 在 Local Host 添加用户账号 如 bill
[[email protected] root] useradd bill
[[email protected] root] passwd bill

3 在Client 上建立用户的公共和私人密钥
以bill登陆到 Local Host 上
[[email protected] local]$ ssh-keygen
Generating 1024-bit dsa key pair
9 o.oOo..oOo.o
Key generated.
1024-bit dsa, created by [email protected] Wed Sep 23 07:11:02 1998
Passphrase :输入一些字符，不超过20个
Again :重复输入一遍
Private key saved to /home/bill/.ssh2/id_dsa_1024_a

4 在Client 上建立用户的认证文件
以bill登陆到 Local Host 上
[[email protected] local]$cd ~/.ssh2
[[email protected] local]$echo "IdKey id_dsa_1024_a" > identification

4. 在 Remote Host 添加用于远程登陆的用户账号 如 bill
[[email protected] root]# useradd bill
[[email protected] root]# passwd bill

5. 以bill登陆到 Remote Host 上
[[email protected] local]$ ssh-keygen
Generating 1024-bit dsa key pair
9 o.oOo..oOo.o
Key generated.
1024-bit dsa, created by [email protected] Wed Sep 23 07:11:02 1998
Passphrase :输入一些字符，不超过20个
Again :重复输入一遍
Private key saved to /home/bill/.ssh2/id_dsa_1024_a

6. 以bill登陆到 Local Host
[[email protected] local]$ cd ~
[[email protected] local]$ cp ./.ssh2/id_dsa_1024_a.pub Local.pub

以bill ftp Remote Host
[[email protected] bill]$ ftp local

ftp> lcd ~/.ssh2
ftp> cd ~/.ssh2
ftp> put Local.pub
ftp> bye

以bill Telnet Remote Host
[[email protected] bill]$ telnet local
[[email protected] bill]$ cd ~/.ssh2
[[email protected] .ssh2]$echo "Key Local.pub" > authorization
